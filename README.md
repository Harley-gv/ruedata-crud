# Instrucciones para ejecutar proyecto de manera local.

## Ejercicio #1

1. Ejecutar el comando para instalar dependencias necesarias.
    ### `npm i` 

2. Ingresar a la pagina [crudcrud.com](https://crudcrud.com)  y copiar el REST enpoint generado.

3. Para ejecutar el proyecto usar el comando 
### ` REACT_APP_API_URL=[tu REST endpoint] REACT_APP_API_ENTITY=pet npm start`
